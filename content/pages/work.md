Markdown Sync is a tiny vanilla pattern for keeping website copy in markdown while shipping plain HTML. Mark the editable regions once, run one Node script, and keep your layout exactly where it belongs.

## Trust

Label: Used By

## Features

### Fast Setup

Title: Markdown source

Write page copy in a small markdown file with frontmatter, sections, lists, and optional fields for labels or links.

### Portable

Title: HTML islands

Add `data-md-*` attributes only where copy should sync. The script updates those islands and leaves classes, layout, assets, and interactions alone.

### Safe Editing

Title: One-way sync

Run markdown to HTML whenever copy changes. Missing required sections fail loudly instead of partially rewriting the page.
